Шаблон js


const App = {
    data() {
        return {
            placeholderString: 'Введите название заметки',
            title: 'Список заметок',
            inputValue: '',
            notes: ['Заметка 1', 'Заметка 2', 'Заметка 3', 'Заметка 4',]
        }
    },
    methods: {
        inputChangeHandler(event) {
            this.inputValue = event.target.value
        },
        addNewNote() {
            this.notes.push(this.inputValue)
        }
    }
}

Vue.createApp(App).mount('#app')

Атрибуты

v-model="variable" - связывает input или другие элементы формы с переменной vue variable

 @keydown.enter="method" - при нажатии на клавиатуре Enter выполняет функцию method

 @click.stop="method" - при клике по элементу выполняет функию method и запрещает всплытие события click дальше по дереву элемтов

 :class="{'active': current === t}" - если current равно t, то элементу добавляется класс active

Вспомогательные фичи

<template v-if="tickers"> - тег template с аттрибутами vuejs не будет появлятся на странице и отображаться в верстке. Можно использовать как вспомогательную обертку для множества элементов под управлением vuejs.
